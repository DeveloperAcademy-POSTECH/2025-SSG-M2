## Description
- 함수처럼 사용할 수 있는 코드 블록
{ (<매개변수 이름>: <매개변수 타입>, ... ) -> <반환타입> in  // head
    // 클로저 표현식 코드(실행될 코드). // 
}

- 이름이 있을 수 있고 없을 수 있다.
```swift
let addition = {(a: Int, b: Int) -> Int in
	return a + b
}
```
- 매개변수 타입 생략 가능 (타입 추론이 가능하기 때문)
```swift
let addition = { (a, b) in 
	return a + b
}
```
- 클로저가 단일 표현식인 경우, return 키워드를 생략 가능
```swift
let addition = { (a, b) in a + b }
```
- 매개변수의 이름도 생략 가능
```swift
let addition: (Int, Int) -> Int = { $0 + $1 }
```
- Ex
```swift
let result = addition(10, 5)
print(result) // 출력: 15
```
